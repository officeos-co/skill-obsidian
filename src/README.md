# Obsidian

Manage an Obsidian vault backed by CouchDB -- read, write, search, graph, tags, templates.

## Categories

- Productivity
- Documents

## Credentials

| Key | Label | Kind |
|-----|-------|------|
| `couchdb_url` | CouchDB URL | text |
| `couchdb_user` | CouchDB Username | text |
| `couchdb_password` | CouchDB Password | password |
| `vault_name` | Vault Name | text |

## Installation

This skill is included in [EnterpriseAgentOS](https://officeos.co) and requires no separate installation.
